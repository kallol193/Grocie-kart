A simple Spring Boot project based on the Grocery inventory management of  a SuperMarket 


                          ---------------FEATURES--------------

MAIN CONTENTS:-

                            each list has 4 subparts:-
                            > id ,
                            > item_name,
                            > item_expDate,
                            > item_grade


OPERATIONS:-
                            capable of operating on penta operations like:-
                            >getItem.
                            >getItemById
                            >postItem
                            >putItEm
                            >deleteItem


APPLICATIONS SPECS:-
                            >Java ver-17
                            >Build -Maven
                            > Refer to rest from POM.XML config file


DEPENDENCIES USED:-
                            >Lombok
                            >Spring MongoDB
                            >Spring web
                            >Spring DevTools


PLATFORM_IN_USE:-
                            >Intellij Idea
                            >MongoDB compass
                            >PostMan
                            >SpringInitializr
